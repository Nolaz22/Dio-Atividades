¿O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar páginas web. Não é uma linguagem de programação, mas sim uma linguagem de marcação que define como o conteúdo é apresentado no navegador. Ela usa tags para organizar elementos como texto, imagens, links e outros conteúdos multimídia em uma página.

  1. Estrutura

    O HTML fornece a estrutura básica de uma página web, definindo como o conteúdo é organizado e apresentado aos usuários.

  2. Tags

    As tags são palavras-chave envoltas em colchetes angulares (ex: p, h1, img). Elas podem ser de abertura (ex: p) e fechamento (ex: /p) para definir o início e o fim de um elemento, ou tags de fechamento único (ex: img src="imagem.jpg").

  3. Atributos

    Atributos fornecem informações adicionais sobre um elemento. Por exemplo, na tag img src="imagem.jpg" alt="Descrição da imagem", src e alt são atributos, src indica a localização da imagem e alt fornece um texto alternativo caso a imagem não possa ser exibida.

  4. Navegadores

    Os navegadores interpretam o código HTML e exibem o conteúdo de acordo com as instruções das tags e seus atributos.

História dos Computadores

A história dos computadores é marcada por uma evolução tecnológica que começou com máquinas mecânicas complexas e culminou nos dispositivos eletrônicos compactos e poderosos que usamos hoje. O desenvolvimento da computação pode ser dividido em gerações, cada uma caracterizada por avanços significativos em termos de componentes, tamanho, velocidade e capacidade.

Clients

Em HTML, o termo "Clients" geralmente refere-se ao navegador web (como Chrome, Firefox, Safari) utilizado pelo usuário para acessar e interagir com páginas web. O cliente envia solicitações ao servidor para buscar informações, exibir conteúdo e realizar ações, tudo dentro do seu ambiente.

Servers

Em HTML, um "servidor" refere-se a um servidor web, que é um sistema computacional responsável por armazenar e entregar conteúdo web aos usuários. Basicamente, é um computador que hospeda sites e seus arquivos (como HTML, CSS, JavaScript, imagens, etc.). Quando você acessa um site, seu navegador envia uma solicitação ao servidor web, que responde com os arquivos necessários para exibir a página.

Estrutura Básica

A estrutura básica de um documento HTML é composta por quatro elementos principais: !DOCTYPE html, html, head e body. A declaração !DOCTYPE html informa ao navegador que o documento é um arquivo HTML5. O elemento html é o elemento raiz que contém todo o conteúdo da página. Dentro de html, temos o head e o body. O head contém metadados sobre o documento, como título, informações de charset e links para arquivos externos (CSS, JavaScript). O contém o conteúdo visível da página, como texto, imagens e outros elementos.

Listas

Em HTML, listas ordenadas e não ordenadas são usadas para apresentar informações de maneiras diferentes. Listas ordenadas ( ol ) usam números ou letras para indicar uma ordem específica dos itens, enquanto listas não ordenadas ( ul ) usam marcadores (como pontos) para indicar que a ordem dos itens não é importante.
